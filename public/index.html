Sobre o Projeto NexIA

NexIA é uma plataforma inovadora que utiliza inteligência artificial para auxiliar médicos na documentação clínica. O sistema transcreve consultas médicas e gera prontuários completos no formato SOAP com códigos CID-10, seguindo as melhores práticas médicas.

Este é um ambiente de desenvolvimento para testes. Algumas funcionalidades estão simuladas para fins de demonstração.
